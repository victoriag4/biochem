#  BioChem Flash

Обучающие карточки по биохимии — веб-приложение на Django для изучения биохимических терминов, аминокислот, ферментов и метаболических путей.

##  Функционал

- Просмотр всех карточек
- Добавление новых карточек
- Редактирование карточек
- Удаление карточек
- Режим тренировки
- Статистика прогресса

## Технологии

- Python 3.x
- Django 6.0
- SQLite
- Bootstrap 5
- HTML/CSS

##  Запуск проекта

1. Клонируй репозиторий

2. Создайте виртуальное окружение
python -m venv venv
venv\Scripts\activate

3. Установите зависимости:
pip install -r requirements.txt

4. Примените миграции 

python manage.py migrate

5. Создать суперпользователя (для доступа к админке)

python manage.py createsuperuser

Введите:

Username: admin (или любой другой)

Email: можно пропустить

Password: придумайте пароль

6. Запустить сервер

python manage.py runserver

7. Открыть в браузере
Перейдите по адресу: http://127.0.0.1:8000
